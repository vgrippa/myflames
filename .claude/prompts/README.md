# Prompts and context used to create myflames

This folder documents the **context and prompts** used when building myflames (with AI assistance). Each file corresponds to a major task or feature. Fill in the **Prompts** and **Outcome** sections with your actual prompts and notes so you can reproduce or share the development workflow.

See the [main README](../../README.md) for project overview and usage.

## Structure of each file

- **Goal** — What you were trying to achieve.
- **Context** — Repo state, key files, and constraints you provided to the AI.
- **Prompts** — The exact prompt(s) and follow-ups that worked.
- **Outcome** — What was implemented and where (files, behavior).
