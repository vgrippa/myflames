Query Analysis panel demos

Each view shows a Query Analysis panel below the chart: optimizer features, warnings (full scans, hash joins, temp tables, filesorts), and suggestions.

  • Hash join (bar) — two full table scans, join_buffer_size suggestion
  • Full table scan (bar) — index suggestion
  • Derived table + sort (treemap) — temp table and sort_buffer suggestions
  • Index condition pushdown (diagram) — optimizer features
